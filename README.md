# DomDom Mobile App

## Description

This is the mobile application for the DomDom project, built with React Native and Expo.

## Features

- Audio streaming
- Meditation timers
- Inspirational quotes

## Installation

To install the dependencies, run:

```bash
npm install
```

To start the app, run:

```bash
expo start
```
